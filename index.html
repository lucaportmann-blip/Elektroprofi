🔌 Block 1 – Grundgrössen der Elektrotechnik

Ladung Q

Definition
Q = I · t  [As = C]
  • Einheit: Coulomb (C = As)
  • Elementarladung: e₀ = 1.602 · 10⁻¹⁹ C
  • Elektron: –e₀  |  Proton: +e₀
  • Gleichnamige Ladungen stossen sich ab

Spannung U

U = W / Q  [V = J/C]
  • Energie pro Ladungseinheit
  • Gerichtete Grösse (Vorzeichen wichtig!)
  • Voltmeter: parallel schalten
  • UAB = –UBA
  • Gefährlich ab ca. 50 V

Strom I & Stromdichte J

I = Q / t = ΔQ/Δt  [A]
J = I / A  [A/mm²]
  • Amperemeter: in Reihe schalten
  • Technische Richtung: positive Ladungen
  • Typisch Hausinstallation: J = 2…10 A/mm²
  • Runder Draht: A = (d/2)² · π

Ohmsches Gesetz & Widerstand R

R = U / I  [Ω]  →  U = R·I  →  I = U/R
Spezifischer Widerstand
R = ρ·l / A  |  ρ = R·A/l  [Ω·mm²/m]
  • Leitwert G = 1/R [S = Siemens]
  • Kupfer: ρ = 0.0178 Ω·mm²/m  |  γ = 56 S·m/mm²
  • Aluminium: ρ = 0.028 Ω·mm²/m

Temperaturabhängigkeit

R(ϑ) = R₂₀ · [1 + α · (ϑ – 20°C)]
R(ϑ) = R₂₀ · (1 + α · ΔT)
  • α (Kupfer, 20°C) = 0.0039 [1/K]
  • Metalle: R steigt mit Temperatur (α > 0)
  • Konstantan: α ≈ 0 (temperaturunabhängig)
  • NTC/Halbleiter: α < 0

Lineare Spannungsquelle

U = Uq – I · Ri
Ri = –ΔU/ΔI = Uq/IK
  • Leerlauf (Ra=∞): U = Uq, I = 0
  • Kurzschluss (Ra=0): IK = Uq/Ri, U = 0
  • Spannungsquelle aus: → Kurzschluss
  • Stromquelle aus: → Unterbruch (Leerlauf)
⚡ Kirchhoffsche Gesetze

1. Kirchhoff – Knotenregel (KCL)

ΣIzu = ΣIab  →  ΣIk = 0
Merke: Alle Ströme die in einen Knoten fliessen = Alle Ströme die rausfliessen.
Zählpfeil zum Knoten → addieren. Weg vom Knoten → subtrahieren.
Beispiel: Knoten A mit I1=300A, I2=−420A, I3=60A, I4=80A
I5 = I1 – I2 + I3 + I4 = 300+420+60+80 = 860 A

2. Kirchhoff – Maschenregel (KVL)

ΣUk = 0  (im Umlauf)
Merke: In einer Masche: Summe aller Spannungen = 0.
Pfeil gleiche Richtung wie Umlaufsinn → addieren, sonst subtrahieren.
Umlaufsinn kann beliebig gewählt werden!
Vorgehen
1. Umlaufsinn festlegen  →  2. Spannungen einsetzen  →  3. Auflösen
🔗 Block 1 – Serieschaltung (Reihenschaltung)
SERIESCHALTUNG
Strom: Überall gleich
I = IR1 = IR2 = … = IRn
Spannung: Teilt sich auf
Utotal = UR1 + UR2 + … + URn
Ersatzwiderstand
Rers = R1 + R2 + … + Rn
Spannungsteiler
U2 = Uq · R2 / (R1 + R2)
Kapazitäten in Serie:
1/Cers = 1/C1 + 1/C2 + … (wie Widerstand parallel!)
Induktivitäten in Serie:
Lers = L1 + L2 + … (wie Widerstände)
Impedanzen in Serie:
Ze = Z1 + Z2 + … (Vektoraddition!)
PARALLELSCHALTUNG
Spannung: Überall gleich
U = UR1 = UR2 = … = URn
Strom: Teilt sich auf
Itotal = IR1 + IR2 + … + IRn
Ersatzwiderstand (allgemein)
1/Rers = 1/R1 + 1/R2 + … + 1/Rn
Zwei Widerstände parallel
Rers = R1·R2 / (R1+R2)
Stromteiler
I2 = I · R1 / (R1 + R2)
Kapazitäten parallel:
Cers = C1 + C2 + … (wie Widerstände in Serie!)
Leitwert: Gers = G1 + G2 + …
Impedanzen parallel:
1/Ze = 1/Z1 + 1/Z2 + …
💡 Tricks: n gleiche Widerstände parallel → Rers = R/n  |  R=0 parallel mit R → Kurzschluss (0Ω)  |  R=∞ in Serie → Unterbruch  |  Vorgehen: immer reine Serie- oder Parallelgruppen identifizieren und vereinfachen
🔀 Block 2 – Netzwerkanalyse

Superposition

Prinzip: Bei mehreren Quellen: jeden Beitrag einzeln berechnen und addieren.

Vorgehen:

  1. Eine Quelle aktiv lassen, alle anderen abschalten
  2. Spannungsquelle ausschalten → Kurzschluss (0V)
  3. Stromquelle ausschalten → Unterbruch (0A)
  4. Ströme/Spannungen berechnen
  5. Wiederholen für jede Quelle
  6. Alle Beiträge addieren (Vorzeichen beachten!)

Schaltung umzeichnen / Ersatzwiderstand

Vorgehen: Alle Bauteile und Knoten identifizieren. Liste der Verbindungen erstellen. Reine Parallel-/Serieschaltungen schrittweise vereinfachen.
Brückenschaltung → weder Serie noch Parallel
→ Knotenanalyse oder Superposition verwenden
Tipp: R = 0Ω ist Kurzschluss (überbrückt alles)  |  R = ∞Ω ist Unterbruch (kein Strom)
⚡ Block 2 – Leistung, Energie & Wirkungsgrad

Elektrische Leistung P

P = U · I = U²/R = I²·R  [W]
W = P · t = U · I · t  [Ws = J]
  • 1 kWh = 3'600'000 J = 3.6 MJ
  • Kosten = W [kWh] · k [Rp/kWh]
  • P positiv → Bauelement nimmt auf
  • P negativ → Bauelement gibt ab

Wirkungsgrad η

η = Pab / Pzu = P2/P1 ≤ 1
Motor: elektrisch → mechanisch
Pmech = U·I·η = M·ω
Mechanische Leistung
P = M · ω  |  ω = 2π·n/60
  • n [min⁻¹] → ω = 2π·n/60 [rad/s]
  • Verlust: Ploss = Pzu·(1–η)

🔋 Block 3 – Kondensator & Kapazität

Definition & Grundformeln

Kapazität (Grundformel)
C = Q / U  [F = As/V]
Plattenkondensator
C = ε · A / d  |  ε = εr · ε0
ε0 = 8.854 · 10⁻¹² As/(Vm)
Gespeicherte Energie
W = C · U² / 2 = Q²/(2C)  [J = Ws]
  • εr Luft = 1  |  Papier = 4–6  |  Glas = 5–10

Strom-Spannungs-Beziehung

i(t) = C · Δu/Δt
Merke: Die Spannung am Kondensator kann NICHT springen!
uC(0⁺) = uC(0⁻)
Am Anfang (Einschalten): verhält sich wie Kurzschluss
→ Strom kann fliessen ohne Spannungsabfall
Im stationären Zustand: verhält sich wie Unterbruch
→ Kein Strom, egal welche Spannung

Schaltung von Kondensatoren

Parallel (wie Widerstände in Serie!)
Cers = C1 + C2 + C3
Serie (wie Widerstände parallel!)
1/Cers = 1/C1 + 1/C2 + 1/C3
Zwei in Serie
Cers = C1·C2 / (C1+C2)

Blindwiderstand (AC)

Kapazitiver Blindwiderstand
XC = 1/(ω·C) = 1/(2π·f·C)  [Ω] ∠–90°
Strom eilt Spannung 90° VORAUS!
XC nimmt mit steigender Frequenz ab.
Bei DC (f=0): XC = ∞ → Unterbruch
🌀 Block 4 – Spule & Induktivität

Definition & Grundformeln

Induktivität (Grundformel)
L = N·Φ / I  [H = Vs/A]
Lange Spule (Näherung)
L = N² · μ · A / l
Gespeicherte Energie
W = L · I² / 2  [J = Ws]
  • μ₀ = 1.257 · 10⁻⁶ Vs/A (Vakuum)
  • μ = μr · μ₀  |  μr Eisen ≈ 500–7000

Strom-Spannungs-Beziehung

u(t) = L · Δi/Δt
Merke: Der Strom durch die Spule kann NICHT springen!
iL(0⁺) = iL(0⁻)
Am Anfang (Einschalten): verhält sich wie Unterbruch
→ Strom kann nicht sofort fliessen
Im stationären Zustand: verhält sich wie Kurzschluss
→ Keine Spannung, egal welcher konstante Strom
Induktiver Blindwiderstand (AC)
XL = ω·L = 2π·f·L  [Ω] ∠+90°
Spannung eilt Strom 90° VORAUS!
XL steigt linear mit der Frequenz.
Bei DC (f=0): XL = 0 → Kurzschluss
⏱ Block 3 – RC-Schaltung: Laden & Entladen
4-Schritte-Methode (immer gleich!): 1. Zustandsvariable: uC(t) – sie kann nicht springen  |  2. Startwert: uC(0⁺) = uC(0⁻)  |  3. Endwert: uC(∞)  |  4. Formel einsetzen

Zeitkonstante τ

τ = R · C  [s]
R = Widerstand der Schaltung aus Sicht des Kondensators
(Spannungsquellen → Kurzschluss; Stromquellen → Unterbruch)

Allgemeine Formel (Ausgleichsvorgang)

Universalformel – gilt für ALLE Grössen
y(t) = y(∞) + [y(0⁺) – y(∞)] · e–t/τ
Beispiel: Laden von uC auf U von 0
uC(t) = U · (1 – e–t/τ)
Ladestrom beim Laden
iC(t) = (U/R) · e–t/τ
Entladen über R (uC(0)=U₀)
uC(t) = U₀ · e–t/τ  |  iC(t) = –(U₀/R)·e–t/τ

RC mit Last (R1, R2, C)

τ = R1‖R2 · C = (R1·R2)/(R1+R2) · C
uC(∞) = U₀ · R2/(R1+R2)
uC(t) = uC(∞) · (1 – e–t/τ)
Merke: τ wird mit dem Thevenin-Widerstand aus Sicht von C berechnet: Quellen kurzschliessen, Widerstand messen.

🌊 Blöcke 5–7 – Wechselstrom & Impedanz

Sinusgrössen & Effektivwert

u(t) = Û · sin(ω·t + φu)
ω = 2π·f = 2π/T  [rad/s]
Effektivwert (RMS)
U = Û / √2  |  I = Î / √2
Netz CH: f = 50 Hz, T = 20ms, U = 230V (Effektivwert), Û ≈ 325V

Impedanz Z (Wechselstromwiderstand)

Z = U/I ∠φZ  [Ω]
φZ = φU – φI
ElementImpedanzWinkelStrom vs Spannung
Widerstand RZ = R0°gleichphasig
Kondensator CZC = 1/(ωC)–90°I eilt U um 90° VORAUS
Spule LZL = ωL+90°U eilt I um 90° VORAUS
Merkregel ELI the ICE man:
EL: Spannung (E) eilt Strom (I) bei Induktivität (L) voraus
IC: Strom (I) eilt Spannung (E) bei Kapazität (C) voraus

Reihenschaltung & Parallelschaltung (AC)

Serie: Impedanzen addieren (Vektoren!)
Ze = ZR + ZC + ZL = R + j(XL–XC)
|Z| = √(R² + (XL–XC)²)
φ = arctan((XL–XC)/R)
Parallel: Admittanzen addieren
Ye = 1/Ze  |  Y = 1/Z

Wechselstromleistung

Scheinleistung S [VA]
S = U · I
Wirkleistung P [W]
P = U · I · cos(φ) = S · cos(φ)
Blindleistung Q [var]
Q = U · I · sin(φ) = S · sin(φ)
Leistungsdreieck
S² = P² + Q²  |  cos(φ) = P/S
ElementPQ
Widerstand RU·I = I²·R0
Kondensator C0–I²·XC (negativ!)
Induktivität L0+I²·XL (positiv)

Phasenverschiebung & Typen

φ = 0° → rein ohmsch → P=S, Q=0
φ = +90° → rein induktiv → P=0, Q>0
φ = –90° → rein kapazitiv → P=0, Q<0
0°<φ<90° → ohmsch-induktiv
–90°<φ<0° → ohmsch-kapazitiv
Phasenverschiebung in Zeit: Δt = φ/(2π·f)
